# R package: bsseq
This is the developer version of Bioconductor package [bsseq](http://bioconductor.org/packages/devel/bioc/html/bsseq.html).  Install in R as:

```r
source('http://bioconductor.org/biocLite.R')
biocLite(bsseq')
```
